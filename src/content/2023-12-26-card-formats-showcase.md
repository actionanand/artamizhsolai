## Gallery of All Card Types

### By Purpose

**For Quotes:**
- Use `::quote` for memorable statements
- Include author attribution when available
- Perfect for inspiration and wisdom

**For Verses:**
- Use `::verse` for poetry, scriptures, and literary excerpts
- Include source/reference for credibility
- Great for cultural and religious content

**For Epics:**
- Use `::epic` for dramatic or historically significant lines
- Add titles for context
- Maximum visual impact for important declarations

**For Palm Leaf Manuscripts:**
- Use `::olai` for ancient wisdom and classical texts
- Perfect for displaying traditional knowledge
- Authentic parchment styling evokes antiquity

**For Newspaper:**
- Use `::newspaper` for news-style content
- Include headlines for article titles
- Classic typography for formal announcements

**For Information:**
- Use `::card[info]` for explanations
- Use `::card[warning]` for cautions
- Use `::card[success]` for achievements
- Use `::card[error]` for problems
- Use `::card[note]` for reminders
- Use `::card[tip]` for advice

## Integration with Other Features

Combine cards with your other formatting options:

- **Text Formatting**: `::color[]`, `::bold{}`, `::combo[]`
- **Tables**: Present data alongside card content
- **Admonitions**: Create attention-grabbing sections
